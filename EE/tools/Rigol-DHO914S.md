Rigol DHO900 Series (DHO914S) Oscilloscope Notes
================================================

Configuration
-------------

Preferred settings (Start menu » Utility):
- Host name: DHO914S_cjs (doesn't stick?)
- Last last: Last
- Input Lock: OFF (when on, press channel buttons 1, 1, 2, 2 to disable)
- Expand: Center (when changing vert. scale)
- Fine/Zoom: Zoom (press horiz. scale knob to get zoom like DS1000 instead
  of fine adjust)

__Quick__ key can be adjusted to several preferences:
- Save Image: defaults to internal storge.
- Save Wave
- Save Setup
- All Measure: brings up list of all measurements for a preconfigured
  channel. Same as chosing Start » Windows » All Measure, selecting source
  channel and pushing Add.
- Reset Statistics
- Waveform Recording
- Save Group

Basic Usage
-----------

Auto measurement requires freq. ≥ 35 Hz and amp. ≥ 10 mV. When activated,
it does its thing and then presents a menu. You can tap:
- Outside the menu: keep current settings.
- Two-cycle waveform: add period and frequency measurements.
- Four-cycle waveform: increase time base and add period/freq measurements.
- Rising/falling edge: trigger at rising/falling edge, decrease time base.
- Back arrow: restore settings from before auto scale.
- Gear: enter Utility » Auto Config menu.
